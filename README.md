# Multimodal Data Analysis

Coursework in computational methods for multimodal data analysis, covering multimodal fusion for visual question answering and spatial transcriptomics clustering.

### Contents

**Part A** — Multimodal data analysis assignment (see coursework brief)

**Part B: PathVQA** — Fusion fine-tuning for visual question answering on pathology images

**SpaGCN** — Spatial gene expression clustering

Each notebook is paired with a PDF report of methodology and results.

### Tools
